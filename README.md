# TacticalHunter - Premium Hunting Gear

A modern e-commerce website built with React, showcasing premium tactical and hunting equipment.

## Features

- Responsive design
- Interactive product showcase
- Animated transitions
- Client testimonials
- Category browsing
- Mobile-friendly interface

## Technologies Used

- React
- Vite
- React Router
- Tailwind CSS
- Framer Motion
- Swiper
- Hero Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
